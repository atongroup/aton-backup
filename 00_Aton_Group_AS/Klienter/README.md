# Klienter — Aton Group AS

## Rekkefølgeregel (obligatorisk)

```
1. NDA signeres
2. Venture Builder-avtale signeres
3. Først da deles nettsider, analyser, søknader og annet Aton-materiale
```

**Aldri del Aton-arbeider før VB-avtalen er signert.**

## Overdragelse av konsept (ved frafall av gründer)

```
1. Overdragelsesavtale signeres mellom Aton og avgående gründer
2. Kjøper identifiseres og fyller inn Vedlegg A
3. Kjøper signerer ny VB-avtale med Aton
4. Aton mottar 20% aksjer i nytt selskap
```
