# Aton Group — Mobil App

**PWA (Progressive Web App)** — installeres direkte fra nettleseren på iOS og Android.

## Filer
| Fil | Beskrivelse |
|-----|-------------|
| `app.html` | Hoved-app (selvforsynt, én fil) |
| `manifest.json` | PWA-manifest for installasjon |
| `icon-192.png` | App-ikon 192×192 |
| `icon-512.png` | App-ikon 512×512 |

## Installasjon på mobil
### iOS (Safari)
1. Åpne `atongroup.no/mobil/app.html` i Safari
2. Del-knapp → «Legg til på hjemskjerm»
3. Appen installeres som native app

### Android (Chrome)
1. Åpne `atongroup.no/mobil/app.html` i Chrome
2. Meny → «Legg til på startskjerm»
3. Eller: banner dukker opp automatisk

## URL på server
Last opp hele `Mobil/`-mappen til `atongroup.no/mobil/`
