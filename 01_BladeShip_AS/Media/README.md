## Pitchvideo — Bruksanvisning

**BladeShip_Pitch_v1.0.html**

1. Åpne filen i Chrome
2. Trykk **F11** for fullskjerm (1920×1080)
3. Videoen spiller automatisk gjennom alle 8 scener (~55 sekunder)
4. Kontroller: → neste scene | ← forrige | R = restart | klikk = neste

For å konvertere til MP4:
- Bruk **OBS Studio** (gratis): Start opptak, spill av HTML-filen i fullskjerm, stopp opptak
- Eller **Xbox Game Bar** (Windows): Win+G → Start opptak
