# 04_Okonomi/SkatteFUNN — [Selskap AS]

SkatteFUNN gir 19% skattefradrag på godkjente FoU-kostnader (lønn, innkjøp, underleverandører).

**Krav:**
- Norsk skattepliktig AS med org.nr.
- FoU-aktiviteter med teknisk usikkerhet
- Revisorattestasjon (skaff revisor tidlig)
- Ingen krav til partner eller pilotkunde

**Innhold i denne mappen:**
- Søknadsskjema (fra skattefunn.no)
- FoU-prosjektbeskrivelse
- Revisorattest
- Årsrapport (etter innvilgelse)

**Frist:** Løpende — send straks etter org.nr.
