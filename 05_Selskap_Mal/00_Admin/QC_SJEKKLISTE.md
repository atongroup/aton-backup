# QC-SJEKKLISTE — Kjøres FØR hver ny versjon leveres

**Obligatorisk. Ingen ny ZIP, nettside, søknad eller partnerbrev uten at alle relevante punkter er sjekket.**

---

## 1. FASIT-verifisering

- [ ] Alle tall i dokumenter/nettsider er hentet fra FASIT — ikke fra hukommelsen
- [ ] Produktbeskrivelse stemmer med FASIT
- [ ] Eierstruktur og aksjekapital er korrekt oppgitt
- [ ] Kontaktinformasjon er korrekt

## 2. Partnerstatus-regler (kritisk)

**Tre gyldige nivåer — ingen andre:**

| Status | Når brukes |
|--------|-----------|
| `Bekreftet (skriftlig)` | Skriftlig ja mottatt |
| `Dialog pågår` | Svar mottatt, ingen forpliktelse |
| `Kontaktet` | Brev/e-post sendt, ingen respons |

- [ ] Ingen partner fremstilles som mer enn det EmailLogg bekrefter
- [ ] Ordene «partner», «samarbeider», «avtale» brukes kun ved skriftlig bekreftelse
- [ ] EmailLogg.md er oppdatert med all utgående kommunikasjon

## 3. Tilskuddspipeline-regler

- [ ] Alle programmer i pipeline er KK-godkjent (se SØKEKRITERIER.md)
- [ ] Ingen søknad sendes uten å ha sjekket: Kan norsk privat AS søke? Kreves partner? Riktig teknologikategori?
- [ ] Ingen programmer som er kjent ikke-kvalifiserte er inkludert

## 4. Nettside-sannhetskrav

- [ ] Ingen påstander om bekreftede partnere uten dekning i FASIT
- [ ] Alle tall på nettsiden er hentet fra FASIT
- [ ] Tilskuddsbeløp og statuser stemmer med FASIT-pipeline
- [ ] Ingen seksjon antyder at noe er fullført som ikke er det

## 5. Dokumenter

- [ ] Idé-NDA signert av begge parter FØR konfidensiell dialog
- [ ] Protokoll korrekt (riktig fra-navn, org.nr., alle signaturer på plass)
- [ ] Aksjonæravtale gjennomgått av advokat før signering
- [ ] Testamente/arveplaner: kun utkast inntil advokat har godkjent

## 6. Teknisk (nettsider)

- [ ] Alle script-blokker er balanserte (ingen JS syntax-feil)
- [ ] Alle `<td>` er balanserte
- [ ] Fargekontrast er tilstrekkelig (hvit tekst på hvit bakgrunn = feil)
- [ ] Mobilvisning er testet — ingen faste kolonner som ikke kollapser

## 7. Versjonshåndtering

- [ ] FASIT.md oppdatert med dato og ny versjon
- [ ] HUSKELISTE.md oppdatert med fullførte oppgaver
- [ ] ZIP bumped til ny versjon
