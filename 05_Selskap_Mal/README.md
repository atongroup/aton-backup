# Selskap AS — Mal v1.0.5
**Aton AdVentures AS · Gründermal**

Denne mappen er standardmalen for alle nye selskaper som tas inn i Aton AdVentures.

## Slik bruker du malen

1. Dupliser denne mappen og gi den nytt navn: `[SelskapNavn]_AS_v1.0.0`
2. Åpne `00_Admin/SESJONSSTARTER.md` — følg stegene
3. Fyll inn alle `[FYLL INN]`-felter i `00_Admin/FASIT.md`
4. Gå gjennom `00_Admin/SØKEKRITERIER.md` — er gründer godkjent?
5. Start med research i `02_Marked/Markedsanalyse/`
