## Konsept
Grunnfelt identifiserer uutviklede tomter med reguleringsposensial, inngår rettighetsavtale
med grunneier uten kostnad for grunneier, og selger rettighetene til boligutbyggere.
Inntektsmodell: 3–8% av salgspris.
