## Bakgrunn

Kremasjonsdelen i Norge er 53,4% (2025) og stiger mot 75% innen 2050. Over 1 100 kremasjoner i Stavanger alene i 2025. Ingen norsk produsent tilbyr luksusurner i premium-materialer.

Velumen har kompetansen, leverandørnettverket og merkevaren til å eie dette rommet.

---

## Juridisk rammeverk

**Jordbegravelse:** Urnen MÅ være av organisk, nedbrytbart materiale. Corten og glass er IKKE tillatt.

**Hjemmeoppbevaring / urnevegg / eksport:** Ingen materialrestriksjoner. Full frihet.

**Velumens strategi:** Alle urner markedsføres og selges for hjemmeoppbevaring og urnevegg-nisjer — aldri for jordbegravelse. Tilbud om biologisk nedbrytbar tilleggsurne for kunder som ønsker jordbegravelse.

## Produkt 1 — Lux Corten-urne

**Posisjon:** Signaturprodukt. Volum. Eksport.

### Designbeskrivelse
Geometrisk kubus i corten-stål A588. Ren form — ingen ornamentikk. Materialet er budskapet. Leveres pre-patinert med varm amber-rust tone direkte fra verksted. Patineringen dypner over tid og eldes med verdighet.

### Tekniske spesifikasjoner
| Parameter | Spesifikasjon |
|-----------|--------------|
| Form | Kubus eller lett avlang boks |
| Utvendige mål | 20×20×20 cm (standard) |
| Innvendig volum | Ca. 215 in³ — romgelig for alle voksne |
| Materiale | Corten-stål A588, 3 mm tykkelse |
| Produksjon | Laserkuttet → brettlagt → TIG-sveist |
| Lokk | Skrugjenger i rustfritt stål, evt. magnetlokk |
| Overflate | Kontrollert pre-patinering (eddik + saltvann) |
| NFC | Valgfritt — NTAG215 chip innlimt i bunn |
| Vekt | Ca. 2,5–3 kg |

### 3 designvarianter
1. **Klassisk kubus** — ren geometrisk boks, ingen pynt
2. **Riflet** — laserkuttede horisontale riller, skaper spill av lys og skygge
3. **Med utskjæring** — enkelt laserkuttet motiv (bølge, trær, abstrakt) i ett panel

## Produkt 2 — Lux Glass-urne

**Posisjon:** Halo-produkt. Bespoke. Høyest pris.

### Designbeskrivelse
Organisk skulptur i lokalt resirkulert glass. Hvert stykke ett-av-ett — glassets flyteegenskaper gir unike fargefordelinger og former som aldri kan reproduseres. Lys lever i materialet. Kan lages med personlig fargeprofil fra avdødes favorittfarger.

### Tekniske spesifikasjoner
| Parameter | Spesifikasjon |
|-----------|--------------|
| Form | Organisk skulptur, bespoke |
| Mål (ca.) | 24–28 cm høy, 14–18 cm diameter |
| Innvendig volum | Ca. 200–230 in³ |
| Materiale | Lokalt resirkulert glass, evt. farget |
| Produksjon | Glassblåsing eller støping hos lokal glasskunstner |
| Lokk | Polert glass eller corten-stål sokkel |
| Overflate | Klar, matt-sandblåst, eller farget |
| NFC | Valgfritt — chip innstøpt i glass under produksjon |
| Vekt | Ca. 2–4 kg avhengig av tykkelse |

### Personaliseringsmuligheter
- Fargepalett basert på avdødes favorittfarger
- Sandblåst motiv (portrett, natur, abstrakt)
- Navn og datoer inngravert
- NFC-minneside med bilder og historier

### Økonomi
| Post | Beløp |
|------|-------|
| Produksjonskost (est.) | kr 3 000–6 000 |
| Presentasjonskasse | kr 500–800 |
| Frakt Europa | kr 300–500 |
| Salgspris | kr 13 000–18 000 |
| Bruttomargin | ~65–70% |

### Markedsføringshook
*«Lys lever i glasset. Som minnene lever i deg.»*

## Produkt 3 — Keepsake-sett

**Posisjon:** Volum. Gave. Familieprodukt.

### Konsept
Sett av 3–5 miniurner i identisk design som Lux Corten. Hele familien kan ha sin symbolske del av asken. Leveres i felles presentasjonskasse av tre eller papp. Kan selges som tillegg til gravminne-bestilling eller alene.

### Tekniske spesifikasjoner
| Parameter | Spesifikasjon |
|-----------|--------------|
| Kapasitet per urne | 10–15 in³ (symbolsk del) |
| Mål per urne | Ca. 7×7×8 cm |
| Antall i sett | 3 (standard) eller 5 |
| Materiale | Corten A588, 2 mm |
| Lokk | Skrugjenger, rustfritt |
| NFC | Ikke inkludert (for lite plass) |
| Emballasje | Felles treplate-kasse med utskjæringer |

### Økonomi
| Post | Beløp (sett av 3) |
|------|------------------|
| Produksjonskost | kr 800–1 200 |
| Presentasjonskasse | kr 300–500 |
| Salgspris | kr 3 200–4 800 |
| Bruttomargin | ~70–75% |

## Produkt 4 — Nisje-front til urnevegg (B2B)

**Posisjon:** Direktesalg til kirkegårder. Ingen konkurrenter.

### Konsept
Tilpassede nisje-fronter i corten-stål eller glass med inngravert navn og datoer + NFC-chip. Kirkegårder som bygger urnevegger trenger fronter — Velumen leverer premium-alternativet til standard granitt.

### Tekniske spesifikasjoner
| Parameter | Spesifikasjon |
|-----------|--------------|
| Mål | Tilpasses nisjens åpning (typisk 22–28 cm bredt) |
| Materiale | Corten A588 eller herdet glass |
| Innhold | Navn, datoer, evt. motiv + NFC-chip |
| Montering | Skruer eller magneter — reversibelt |
| NFC | Inkludert standard |

### Økonomi
| Post | Beløp |
|------|-------|
| Produksjonskost | kr 800–1 800 |
| Salgspris | kr 5 000–12 000 |
| Bruttomargin | ~75–85% |

### Salgsstrategi
- Kontakt kirkegårder direkte — ikke via begravelsesbyråer
- Stavanger kirkelige fellesråd er første kontaktpunkt
- Rogaland har allerede 2 urnevegger i drift (Sola, Bergen) og flere planlagt

## Neste steg (prototyper)

1. Be metallverksted i Stavanger om pris på corten-kubus 20×20×20 cm, 3mm, TIG-sveist
2. Finn glassblåser i Rogaland/Vestland for glass-urne prototype
3. Test NFC (NTAG215) innlimt i bunn av corten-boks
4. Avklar med urnevegg-kirkegårder om interesse for nisje-fronter
5. Design presentasjonskasse til Keepsake-sett
