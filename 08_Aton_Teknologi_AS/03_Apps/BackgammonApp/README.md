# BackgammonApp — Aton Teknologi AS
**Status:** 🆕 Under bygg (hobbyprosjekt/PoC)
**Opprettet:** 6. juni 2026

## Konsept
Webbasert backgammon-spill for mobil hvor venner enkelt kan spille mot hverandre, med rankingliste.

## Tech-stack (planlagt)
- React (webbasert — ingen App Store-prosess)
- Supabase Realtime (sanntid, begge ser brettet live)
- Supabase Auth (brukerkontoer + rankingliste)

## MVP-mål
- [ ] Backgammon-brett visuelt (SVG/Canvas)
- [ ] Spillogikk (lovlige trekk, terning, hitting, bearing off)
- [ ] Lobby: lag spill → del lenke → venn kobler på
- [ ] Sanntid via Supabase Realtime
- [ ] Rankingliste

## Faser
| Fase | Innhold | Estimat |
|------|---------|---------|
| 1 | Brett + spillogikk | 1–2 sesjoner |
| 2 | Multiplayer/sanntid | 1 sesjon |
| 3 | Ranking + brukerkontoer | 1 sesjon |
